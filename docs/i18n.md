# Sistema Multilíngue (i18n) - Concluído! 🎉

O suporte dinâmico a múltiplos idiomas foi totalmente integrado ao aplicativo. Agora, não apenas o painel de configuração possui idioma selecionável, como toda a interface responde nativamente a opção inicializada! 🌐

## Mudanças Realizadas
1. **Configuração Persistida:** Opção adicionada na nova aba `General` das configurações, permitindo selecionar o idioma usando o cursor. Toda mudança é salva automaticamente dentro da classe de configuração para garantir compatibilidade com `config.toml` e até legados `config.txt`.
2. **Refatoração da Interface (`gui.py`):** Todos os textos absolutos ("hardcoded") escritos em inglês foram programaticamente substituídos pela função dinâmica tradutora `t("chave_da_frase")`.
3. **Módulo i18n Nativo:** O arquivo `i18n.py` que estava disperso na raiz principal do projeto agora reside no lugar correto dentro de `dsno_processor/i18n.py` respeitando todos os módulos de importação.


> [!NOTE]
> Quando um usuário altera a seleção de idioma no painel de Configurações e clica em **Save**, o novo valor ficará em vigor imediato e o painel solicitará a reinicialização da interface gráfica principal para carregar as substituições ao arrancar novamente!

## 🛠️ Como adicionar Traduções Posteriormente
Conforme solicitado na sua pedida original, aqui estão as explicações sobre como editar as traduções no futuro, caso você queira refinar um texto ou expandir para novos idiomas.

Abra o arquivo fonte localizado em [`dsno_processor/i18n.py`](file:///c:/Users/ao32v/repositories/dsno-processor/dsno_processor/i18n.py).

### 1. Editar um Texto Existente
Sempre que quiser alterar uma frase, procure pelo dicionário local `_TRANSLATIONS`. Ele possui suporte para os idiomas ativos divididos por códigos (`en` e `pt`).

Por exemplo, se quiser alterar o texto do botão de Cancelar e a mensagem for `"Cancel"`:
```python
# Em _TRANSLATIONS["en"]
"btn.cancel": "Cancel",

# Em _TRANSLATIONS["pt"] 
"btn.cancel": "Cancelar",
```
Basta trocar o lado direito destas aspas no respectivo idioma e reiniciar o seu painel de execução!

### 2. Adicionar Tradução a um Botão Novo
Se por acaso você codificar e adicionar uma nova aba, menu, ou botão novo no projeto de painel (no `gui.py`), eis o processo listado:

**No arquivo `i18n.py`:**
1. Crie uma chave nova e descritiva dentro de *ambas* as categorias globais:
```python
# em inglês
"minha_aba.botao_de_exportar": "Export to PDF",

# em português
"minha_aba.botao_de_exportar": "Exportar p/ PDF",
```

**No arquivo `gui.py`:**
2. Use essa chave envolta pela função de tradução (você verá que importações pra função `t` já estão prontas no começo do script `from dsno_processor.i18n import t`):
```python
ctk.CTkButton(
    aba, 
    text=t("minha_aba.botao_de_exportar"), 
    command=self.minh_func
# ...
```

### 3. Como Adicionar Múltiplos Formatos (f-strings)
Note que variáveis f-strings antigas como `f"Baixado: {count}"` foram mudadas para respeitar localizações. Para preenchimentos dinâmicos a serem calculados num processo, crie sua representação `i18n.py` entre chaves assim:
```python
"msg.contador": "Concluído com: {processados} processos de {totais}.",
```
E use na inferência `gui.py` usando simples vírgulas:
```python
texto_final = t("msg.contador", processados=meus_processados, totais=limitador)
```

### 4. Como Adicionar **Espanhol** ou um Idioma Inteiramente Novo
Para adicionar novos pares de idiomas posteriormente, como o espanhol (`es`):

1. Modifique a chave `SUPPORTED_LANGUAGES`:
```python
SUPPORTED_LANGUAGES: dict[str, str] = {
    "en": "English",
    "pt": "Português",
    "es": "Español",   # <<-- Adicione aqui
}
```
2. Adicione os dicionários clonados no enorme bloco `_TRANSLATIONS`:
```python
_TRANSLATIONS: dict[str, dict[str, str]] = {
    "en": { ... },
    "pt": { ... },
    # ══════════════════════════════════════════════════════════════
    # ESPAÑOL
    # ══════════════════════════════════════════════════════════════
    "es": {
       "app.title": "Procesador DSNO",
       "btn.save": "Guardar",
       # ...
    }
```
Isso é tudo! A aba respectiva "General" das próprias configurações do App vai automaticamente descobrir essa sigla e prover na combobox do menu para seu usuário sem que você tenha de tocar no componente da UI. 🇪🇸
